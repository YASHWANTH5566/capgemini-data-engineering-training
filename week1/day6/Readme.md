# 🚀 Mini & Advanced Pipeline -- Car Sales Data (SQL + PySpark)

## 📌 Objective

Build an end-to-end data pipeline using **PySpark and SQL** focusing
on: - Data cleaning - Validation - Transformations - Analytical insights

------------------------------------------------------------------------

## 🗂️ Dataset Tables

1.  customers (customer_id, name, city)\
2.  cars (car_id, brand, model, price)\
3.  sales (sale_id, customer_id, car_id, sale_date, quantity)\
4.  dealers (dealer_id, name, city)\
5.  sales_dealer (sale_id, dealer_id)

------------------------------------------------------------------------
