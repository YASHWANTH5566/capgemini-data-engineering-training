### 🚀 Phase 4A – Bucketing & Segmentation in PySpark

## 📌 Objective
Learn how to convert continuous data into meaningful categories (segmentation) using PySpark.

---

## 🧠 Core Concept

Bucketing = Converting numerical values into categories

Example:
- Gold
- Silver
- Bronze

---

## ⚙️ Methods Implemented

## 1. Conditional Logic
- Most common
- Business rule based

## 2. SQL CASE
- SQL equivalent of conditional logic

## 3. Bucketizer (MLlib)
- Used in ML pipelines

## 4. Quantile-based Segmentation
- Based on data distribution

## 5. Window Functions
- Ranking using percent_rank()

---

## 📊 Tasks Completed

- Customer segmentation (Gold/Silver/Bronze)
- Count customers per segment
- Quantile-based segmentation
- Comparison of methods

---

## 🔁 ETL Flow

Extract → Clean → Aggregate → Segment → Analyze

---

## 🧠 Key Learnings

- Continuous data needs categorization
- Different segmentation techniques exist
- Choice depends on business needs
- Quantiles adapt better to changing data
